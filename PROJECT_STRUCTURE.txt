PAGES & ROUTES
==============

Public Routes:
--------------
/                       → Homepage (Hero, Features, Services, Process)
/about                  → About page (Story, Values, Brands)
/services               → Services listing (12 service cards)
/services/[slug]        → Individual service detail pages (12 pages)
/gallery                → Before/after work gallery
/blog                   → Blog posts listing
/contact                → Contact form & information

TECH STACK
==========

Framework:     Next.js 14 (App Router)
Language:      TypeScript
Styling:       Tailwind CSS v4
Icons:         Lucide React  
Deployment:    Vercel
Build Tool:    Turbopack

SCRIPTS
=======

npm run dev      → Start development server (http://localhost:3000)
npm run build    → Build for production
npm start        → Start production server
npm run lint     → Run ESLint

2. Develop:        npm run dev
